# 🏠 Home Automation & Interior Design Website
### Business Analysis Project

## 📌 Project Overview

This project is a Business Analysis case study for a Home Automation & Interior Design platform. The system connects customers, home automation companies, interior design partners, and administrators through a centralized web application.

The project focuses on gathering business requirements, defining user stories, documenting functional and non-functional requirements, and designing a scalable solution.

---

## 🎯 Objectives

- Streamline customer inquiry management
- Enable consultation booking
- Manage home automation services
- Support affiliate referral management
- Improve communication between customers and service providers
- Provide centralized administrative control

---
